🎬 Netflix Clone - React & Vite
A high-fidelity Netflix landing page clone that focuses on smooth user transitions and precise UI replication. This project was built to master React hooks, CSS animations, and asset management within the Vite ecosystem.

🚀 Project Overview
This application replicates the modern Netflix experience, beginning with a premium video intro animation that seamlessly transitions into a functional login interface and a "Who's watching?" profile selection screen.

🛠️ Tech Stack
Frontend Library: React.js

Build Tool: Vite

Styling: CSS3 (Flexbox, Grid, Keyframe Animations)

Design Tool: Figma

✨ Key Features
Cinematic Intro: A smooth onEnded video event that triggers a fade-out transition to the main UI.

Dynamic Login Card: \* Responsive layout with adjustable transparency.

Interactive password visibility toggle using SVG eye icons.

Form validation ready for future backend integration.

Profile Selection: An authentic "Who's watching?" screen featuring a 2x2 grid, SVG avatars, and hover effects.

Asset Optimization: Uses Vite's asset pipeline to handle high-resolution backgrounds and video files.
